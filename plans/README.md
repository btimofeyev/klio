## Findings considered and rejected

- Native iOS/Android app first: rejected for the MVP. A responsive PWA supports mobile photo capture and desktop planning with substantially less delivery and billing complexity.

- Browser automation as the only gradebook integration: rejected. Prefer a vendor API or parent export; use browser automation only after establishing a secure, supervised fallback.
